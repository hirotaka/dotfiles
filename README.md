# dotfiles

## Setup

### Xcode

Install Xcode.

```bash
xcode-select --install
```

### brew

Run following command on terminal.

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/hirotaka/dotfiles/master/install.sh)"
```
